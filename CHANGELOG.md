## [1.0.0] - 2026-01-11

### Added

- Initial release
- Chat interface with DeepSeek AI model via OpenRouter API
- Real-time message streaming
- Beautiful UI with animations
- Markdown support for AI responses
- Loading states with shimmer effects
- Connectivity check
- Dark/Light theme support
- Splash screen with animations
- Error handling and retry mechanisms

### Features

- **AI Integration**: Uses DeepSeek AI through OpenRouter API
- **Modern UI**: Material Design with smooth animations
- **Markdown Rendering**: Support for formatted AI responses
- **Responsive**: Works on Android, iOS, Web, Windows, Linux, and macOS
- **Secure**: API key handling through environment variables
